## Custom Terminal Prompt

For an additional personal touch, you can customize your terminal sessions with a "yourname@computer $" prompt:

```sh
export PS1="\[\033[36m\]yourname\[\033[m\]@\[\033[32m\]computer\[\033[m\]\$ "
```
